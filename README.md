# Shraddha — Tech Monorepo

Private repository containing the Shraddha app and its supporting backends.

## Structure
- `app/` — Flutter mobile app (module: shanti)
- `backend/` — main API backend (Python / FastAPI, managed with uv)
- `face-swap-backend/` — face-swap microservice (Python)

Build artifacts, virtual environments, and secrets are git-ignored.
See each subfolder's own `README.md` for setup instructions.
